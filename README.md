swapBlock - Binds triggers and blocks for swapping
---------------

## SYNOPSIS

    $.swapBlock.bind('click', $(".triggers"), $(".blocks"))
    $.swapBlock.bind('click', $(".triggers"), $(".blocks"), {...})
